# NabraAI 🎙️

**NabraAI** is an intelligent, AI-driven interview coaching application designed to help users improve their interview performance by analyzing vocal emotions and providing actionable feedback.

## 🚀 About The Project

Preparing for professional interviews can be challenging. NabraAI acts as a personal interview coach, leveraging advanced artificial intelligence to detect and analyze the user's emotional state through their voice tone. By providing insights into vocal delivery, the application empowers candidates to communicate with greater confidence and clarity.

## ✨ Key Features

- **Vocal Emotion Detection:** Accurately identifies emotions from voice using state-of-the-art AI models.
- **Real-Time Feedback:** Offers constructive coaching based on vocal analysis to improve delivery.
- **Cross-Platform Experience:** Seamless and intuitive user interface built for multiple platforms.
- **Secure & Fast Processing:** Efficient backend architecture to ensure quick response times during practice sessions.

## 🛠️ Tech Stack

This project was built using a robust and modern technology stack:

- **Frontend:** [Flutter](https://flutter.dev/) — For a responsive and beautiful cross-platform UI.
- **Backend:** [FastAPI](https://fastapi.tiangolo.com/) — For high-performance, asynchronous API routing.
- **Machine Learning:** [Wav2Vec 2.0](https://huggingface.co/docs/transformers/model_doc/wav2vec2) — For speech processing and accurate emotion recognition.

## 🏁 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Flutter SDK
- Python 3.x
- Git

### Installation

1. Clone the repository:

```sh
git clone https://github.com/amenh19985066-eng/NabraAI.git
```

2. Navigate to the project directory:

```sh
cd NabraAI
```

3. Install Flutter dependencies:

```sh
flutter pub get
```

4. Run the app:

```sh
flutter run
```
